Advanced Display

Description

The Advanced Displays are specially designed displays for trains and stations. They can show a lot of information in a beautiful design. There are many settings that create various applications, leading to a better create train experience.
There are 4 different types of Advanced Displays:
1. Advanced Display
2. Advanced Display Block
3. Advanced Display Panel
4. Small Advanced Display

To craft the display, you'll need a crafting table with 4 Andesite Alloys, 2 Sturdy Sheets, and 3 regular Display Boards. This recipe yields 6 Advanced Displays directly.

Explanation

You can configure the displays with a wrench. There are multiple types in which the Advanced Display can be used. This is the main page of the Advanced Display Settings. On the bottom left there is a settings button for the global settings.

There are 3 options settings you can change. The middle one is the Information type. That’s just the setting to determine how much information should be shown on the display. For this, there are three options available. Simple is not very detailed, and Informative has the most information.

On the Display Settings, the first one is Display Type. This determines what information is shown on the Advanced Display. There are also three options available.

The first one, ‘Train Destination,’ in the Simple option, shows the name of the train and the next destination of the schedule or the current schedule title.

The Detailed one shows the same as the Simple one, but in addition to this, it shows the stops the train is attempting until the last stop of the schedule.

The Informative one shows the same as the Detailed one, but on top of that, it shows the last stop in the schedule and the carriage number.

The second option at the Display Type section is Passenger Information. In simple, it shows the next stop with the exit direction when driving, and when the train stopped at a station, it shows the current stop name. It also shows the train name and the end stop on longer rides between stations.

On Detailed, it shows the same as the Simple does, but also the time and speed while riding the train.

On Informative, it shows everything the Detailed did but all at the same time, and it shows the stations between the start and the end.

The last option in the Display Type section is Platform Display. When set to Simple, it shows the current in-game time, and when a train is approaching the station, it shows the train name with the time the train is departing, and when set in the global settings, the platform number.

On Detailed, it shows the current time and an overview of the train with departure, the name of the train, and the schedule title or last station in the schedule.

On Informative, it shows the next train with the name, departure, and schedule title or last station, but also the stations between the current station and the last station. Below this, there is a list of following trains that are approaching this train station.

In the Advanced Display Settings, the last option is just to determine on which side the display should show the information - on the front, the back, or both sides. This works on all advanced displays.
