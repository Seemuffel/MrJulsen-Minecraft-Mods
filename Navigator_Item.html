Navigator Item

Description

The navigator is a handy item that you carry in your inventory. Similar to a mobile phone, it allows you to check schedules, search for connections, verify delays, and organize your travels. With its user-friendly interface, the navigator makes train travel a straightforward experience.

The item is crafted in a Mechanical Crafter. You need 5 Iron Sheets, 4 Andesite Alloys, 1 Copper Sheet, 1 Glass Pane and 1 Precision Mechanism.

Explanation

This is the main page of the Create Railway Navigator. Here you can set a starting point and an endpoint and then search for trains that travel this route. The button in the top right automatically sets the currently nearest station. Below that button is the search button. Below that, the trains that travel between are displayed. Transfers are also displayed depending on the route.

The button on the bottom left takes the user to the search settings. There, you can set how long you want the minimum transfer time to be when the navigator searches for a route where you have to change trains. Additionally, you can select train categories in the train filter that you want to be displayed. These can be changed in the global settings.

In the Global Settings, you can adjust all Navigator and Display-specific configurations. Firstly, in the Train Station Tags section, you can specify which stations belong to a particular station. For instance, if there are two platforms in a station with different names, you can group them under one name and input the platform numbers. On the Train Station Blacklist, you can enter stations that shouldn't be displayed on the displays and in the navigator. With Train Groups, you can determine which trains are grouped together, giving them a unified name. Lastly, the Train Blacklist allows you to specify which trains should not be displayed, such as freight trains or test trains.
The global settings are synchronized on a server for all players.
